HTTP报文

http报文由三部分组成：

  • start line (起始行)
  • header（首部块）
  • body（数据主体，这一部分为可选）

起始行和首部是由行分隔的ASCII文本，每一行以回车符（ASCII码13）和换行符（ASCII码10）结束；

1.3 状态码

状态码范围 分类
100 ～ 199 信息提示
200 ～ 299 成功
300 ～ 399 重定向
400 ～ 499 客户端错误
500 ～ 599 服务器端错误

1.4 方法

方法 描述
GET 从服务器获取一份文档
HEAD 与GET方法行为类似，但服务器在应答报文中只返回头部信息
POST 向服务器输入数据,用来创建资源，不具有幂等性
PUT 向服务器写入文档，用来更新资源，具有幂等性
TRACE 对可能经过代理服务器传送到服务器上的报文进行追踪
OPTIONS 获取服务器支持的方法类型
DELETE 从服务器上删除一份文档

1.5 HTTP和HTTPS

HTTP就是HTTP over TCP over IP，其安全版本HTTPS就是在HTTP和TCP之间插入一个密码加密层。

HTTPS网络协议栈 所在层
HTTP 应用层
TSL或SSL 安全层(HTTP没有)
TCP 传输层
IP 网络层
网络接口 数据链路层
信号和介质 物理层

每个TCP段都是由IP分组承载，从一个IP地址转发到另一个IP地址，每个IP分组中都包括：

  • 一个IP分组首部（通常为20字节）

  • 一个TCP段首部（通常为20字节）

  • 一个TCP数据块（0个或多个字节）
